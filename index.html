About me

Saya adalah lulusan S1 Program Studi Pendidikan Geografi Universitas Tadulako, dalam keilmuan Geografi saya tertarik pada bidang keilmuan Kartografi, Penginderaan Jauh dan Sistem Informasi Geografis seperti dalam mengaplikasikan sofware GIS diantaranya ArcGis/ArcGis Pro, QGIS, GlobalMapper, Google Earth Engine, ERDAS IMAGINE dan WebGIS membuat sistem informasi data spasial berbasis website serta mengoperasikan alat survei pemetaan lainya seperti GPS, Kompas, Theodolite dan juga Drone khusus pemetaan, saya memiliki 3 HAKI (Hak Kekayaan Intelektual) di bidang SIG dan Pemetaan.

Sebagai seorang berlatar belakang sarjana pendidikan saya juga menyukai ilmu pengetahuan yang berkaitan dengan pengajaran seperti pengetahuan Geografi secara umum dan pengetahuan lainya di antaranya Sejarah, Sosiologi, Politik, Agama, Psikologi dan Filsafat.

Sebagai seseorang yang fokus mendalami bidang teknis keilmuan Geografi saya pernah magang MSIB Kampus Merdeka di Kementrian ATR/BPN penempatan dinas Kabupaten Gowa Sulawesi Selatan sebagai tim penyusun RDTR melakukan penyusunan buku fakta dan analisis, survei lapangan dan mengelola data spatial pembuatan peta dasar dan peta tematik serta pengelolaan data lanjutan lainya.

Saya juga tertarik mempelajari bidang keilmuan teknis lainya seperti programming dengan mempelajari bahasa pemrograman seperti Java Script, PHP, Python dan bahasa markup HTML dan CSS dan framework lainya seperti Codeigniter dan Laravel dan saya sementara mendalami keterampilan dibidang GeoAI menggunakan Machine Learning dan Deep learning. Selain itu saya juga mempunyai keterampilan lainya seperti Desain Grafis menggunakan (Adobe Photoshop/Illustrator dan CorelDraw) serta dapat membuat model 3D menggunakan Sketchup.

saya juga mempunyai kemampuan softskill seperti teamwork, problem solving, komunikasi, public speaking, negosiasi dan bahasa inggris.
